# 如何贡献 / 建议

  1. Pull request

    1. Fork 后，进行修改。
    2. 发起 Merge 请求。

  2. 邮件提交

    1. 发送邮件到 support__at__mail.sustech.online (__at__ 替换为 @)

  3. 代理提交

    1. 进入南科手册 QQ群 (opens new window): 905618908
    2. 提交规范编写的Word，Markdown 文件给管理员。
    3. 管理员整理上传。
